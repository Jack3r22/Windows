# Impedir la instalación de software.


1. Nos dirigimos a **Configuración del equipo > Directivas > Plantillas administrativas > Componentes de Windows > Windows Installer > Desactivar Windows Installer**
2. Marcamos la opción **“Habilitada”** y seleccionamos en el desplegable **“Solo para aplicaciones no administradas”** . Esta ultima opción permite a los usuarios instalar lo programas que el administrador ofrezca en el escritorio o publique (Agregar o quitar programa)
